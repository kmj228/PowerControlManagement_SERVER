
## 사전 요구사항

- [Node.js](https://nodejs.org/) 18 이상
- [MariaDB](https://mariadb.org/) (로그 기능 사용 시)

## 실행 방법

```bash
npm install
node server.js
```

서버가 시작되면 콘솔에 접속 주소가 출력됩니다

```
─────────────────────────────────────────────────
  서버가 시작되었습니다. 아래 주소로 접속하세요.
─────────────────────────────────────────────────
  로컬    : https://localhost:3000
  이더넷  : https://192.168.x.x:3000
─────────────────────────────────────────────────
```

## 첫 실행 안내

1. 브라우저에서 `https://localhost:3000` 접속
2. 초기 계정으로 로그인
   - 아이디: `admin` / 비밀번호: `admin1234`
   - 로그인 후 반드시 비밀번호를 변경하세요
3. 로그인 후 시작 가이드에 따라 순서대로 설정
   - **DB 설정** — 설정 메뉴 → DB 설정에서 MariaDB 접속 정보 입력
   - **장비 등록** — 장비 목록 → 추가 버튼으로 WIZ550S2E 등록
   - **인증서 설치** — 인증서 설치 버튼으로 HTTPS 경고 제거

## 주요 기능

- WIZ550S2E TCP 연결 및 채널별 전원 ON / OFF / RESET 제어
- 실시간 상태 모니터링 (WebSocket, 10초 주기 STATUS 수신)
- 연결 상태 4단계 표시 — 연결됨 / 타임아웃 / 연결 해제 / 미연결
- MariaDB 로그 기록 / 검색 / CSV 내보내기
- 다중 사용자 및 역할 구분 (관리자 / 일반 사용자)
- HTTPS 자동 인증서 생성 (openssl 또는 순수 Node.js)
