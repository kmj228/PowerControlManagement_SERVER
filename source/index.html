접속하는 모든 PC에서 한 번씩 설치해 주세요.

1
아래 버튼으로 인증서 파일을 받아 주세요.
2
받은 DeviceManager-CA.crt 파일을 더블클릭해 주세요.
3
인증서 설치 → 로컬 컴퓨터 → 모든 인증서를 다음 저장소에 저장 → 찾아보기 → 신뢰할 수 있는 루트 인증 기관 선택 → 확인 → 다음 → 마침을 해 주세요.
4
브라우저를 완전히 닫았다가 다시 열고 연결해 주세요.
Firefox 사용자는 브라우저 설정 → 인증서 보기 → 인증 기관 → 가져오기에서 별도로 설치해 주세요.
